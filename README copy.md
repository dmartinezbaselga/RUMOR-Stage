# RUMOR Stage Simulator

This package package is an adpatation of the open source Stage simulator to be used with reinforcement learning, and it is adapted to run with RUMOR. Please, go to [the main repo of the project](https://github.com/dmartinezbaselga/RUMOR) for more information and installation instructions.
